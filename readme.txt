== Description ==

 The Helios Solutions WooCommerce Hide Price and Add to Cart button plug-in is made to remove an woo commerce functionality providing a perfect platform for products which aren’t currently for sale online. The plug-in actually hides the price/add to cart button from   the product image. 
 The plug-in also provides an option to restrict it to the users who are logged in, rather than taking the whole store offline. Which means, the price and add to cart button won’t be visible for the users those are not logged in.


= Features =

-Woo commerce hide product price and add to cart button   
-Easily customizable using the admin panel options
-Offers an ability to hide prices or show prices to both logged-in and normal users
-Can disable ecommerce functionality using this plug-in by simply disabling the cart


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin folder `Helios solutions woocommerce hide price and add to cart button` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins -> Installed Plugins -> Activate Helios Solutions WooCommerce Hide Price and Add to Cart button" menu in WordPress.
3. Find Woo commerce Helios solutions hide price and add to cart button plugin ,which is located From  woo commerce menu -> woo commerce setting-> visibility tab options.
